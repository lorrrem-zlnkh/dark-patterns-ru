# Спецификация каталога обманных паттернов

Версия структуры, согласованная с заказчиком. Визуал — отдельно и позже.

## 1. Таксономия

### Слой 1 — механизм (6 групп, смысловой костяк)
1. **Сокрытие информации** — важное прячут или показывают слишком поздно
2. **Препятствия** — нужное действие искусственно усложняют
3. **Давление** — торопят и пугают упущенной выгодой
4. **Принуждение** — желаемое связывают с нежелательным
5. **Подмена смысла** — текст вводит в заблуждение
6. **Интерфейсные уловки** — манипуляции вёрсткой и значениями по умолчанию

### Слой 2 — теги «где встречается» (сферы)
🏦 Банки · 🛒 Маркетплейсы · ▶️ Подписки и стриминги · 🚕 Доставка и такси · 📞 Операторы связи · 📱 Игры и приложения · 🌐 Соцсети и контент · 🏛 Госсервисы

Один паттерн = **1 механизм** + **несколько сфер**. Каталог режется двумя способами: «что это за приём» и «где я с этим столкнусь».

## 2. Функционал (v1 — средний объём)
- 🔎 Поиск по названию/описанию
- Фильтр по механизму (6 групп)
- Фильтр по сфере (теги)
- Карточка → страница статьи
- Блок «Связанные паттерны» внизу статьи
- *(отложено)* уровень вреда, форма «прислать пример», личный кабинет

## 3. Шаблон статьи (наполнение)
1. Краткое определение (выноска-цитата)
2. **Что это**
3. **Как работает**
4. **Где встречается в СНГ** — пример завуалированно, на основе открытых источников
5. **Чем вредит**
6. **Как распознать и защититься**
7. **Что говорит закон РФ** — ЗоЗПП, 152-ФЗ «О персональных данных», 38-ФЗ «О рекламе», практика ФАС, ЦБ РФ, Роспотребнадзора
8. **Связанные паттерны**

## 4. Принципы текста
- Уникальный рерайт, не похожий на deceptive.design
- Приземление на реалии продуктов СНГ
- Примеры — аккуратно, без прямых обвинений: «такой приём встречается у…»
- Названия продуктов — завуалированно (см. глоссарий)

## 5. Как называть продукты в примерах

**Политика (обновлено 2026-06): говорим АБСТРАКТНО.** Никаких конкретных компаний и никаких цветных завуалированных названий («Зелёный банк», «Розовый маркетплейс» и т.п. — отменены). Используем только обобщённые категории.

| Сфера | Как называть |
|---|---|
| Банки | крупный банк, банковское приложение |
| Маркетплейсы | крупные маркетплейсы, маркетплейс |
| Операторы связи | сотовые операторы |
| Такси/доставка | агрегаторы такси, сервисы доставки |
| Авиа | лоукостеры, чартерные перевозчики |
| Соцсети/контент | соцсети, новостные ленты |
| Почта | почтовые сервисы, личные кабинеты |
| Подписки | музыкальный стриминг, видеосервисы |
| Мессенджеры | популярный мессенджер |
| Приложения | сайт-каталог программ, мобильные игры |

### Тон примеров
- Всегда «такой приём встречается у…», без прямых обвинений и без узнаваемых деталей (слоганов, цветов бренда, логотипов).
- Каждая статья содержит дисклеймер: *«по открытым публикациям, распространённые практики, а не доказанные нарушения конкретных компаний»*.
- Источники (Habr, vc.ru, РБК, Skillbox, Роскачество, дела ФАС/ЦБ) — для внутренней проверки фактов; в текст НЕ выносим.
