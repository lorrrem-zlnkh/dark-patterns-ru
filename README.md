# Обманные паттерны

Русскоязычный каталог обманных паттернов интерфейсов (deceptive / dark patterns) на примере продуктов СНГ. Просветительский проект: что это за приёмы, где встречаются, чем вредят, как распознать и что говорит закон РФ.

**27 паттернов** в 9 группах по принципу действия. Дизайн — в стиле [shadcn/ui](https://ui.shadcn.com): нейтральная палитра, светлая и тёмная темы.

## Стек

Статический сайт без зависимостей. Контент — Markdown в `content/patterns/`, генератор — `build.mjs` (Node), вывод — в `dist/`.

## Локальный запуск

```bash
node build.mjs      # собрать сайт в dist/
node serve.mjs      # поднять локальный сервер на http://localhost:4321
```

При правке контента или стилей пересоберите (`node build.mjs`) и обновите вкладку.

## Структура

```
content/patterns/   статьи (Markdown + frontmatter)
assets/styles.css   стили (дизайн-токены shadcn/ui)
build.mjs           генератор статики
serve.mjs           локальный сервер для предпросмотра
docs/               спецификация каталога и глоссарий
```

## Публикация

Сайт автоматически собирается и публикуется в GitHub Pages при каждом пуше в `main` (workflow `.github/workflows/deploy.yml`). Базовый путь для проектного сайта задаётся переменной `BASE_PATH` в CI.
